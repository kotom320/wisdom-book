# 지혜의 서 (Wisdom Book)

종교색 없이, 고전의 지혜를 현대의 언어로.

## About

출근길 지하철에서 한 장씩 펼쳐보는 '지혜의 서'. 성경, 불경, 동양 고전, 서양 철학에서 길어 올린 지혜를 현대적 해석과 함께 매일 한 구절씩 전해드립니다.

## Tech Stack

- **Framework**: Next.js (App Router) + TypeScript
- **Styling**: TBD
- **Infra**: AWS S3 + CloudFront
- **CI/CD**: GitHub Actions
